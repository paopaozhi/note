## tar命令

```sh
tar [选项] 源文件或目录
```

### **简介**

1.打包解包**.tar**文件

2.打包解包**.tar.gz**文件 --基于1添加选项**-z**

3.打包解包**.tar.bz2**文件 --基于1添加选项**-j**

### **选项**

打包命令

| 选项 | 含义                                                         |
| ---- | ------------------------------------------------------------ |
| -c   | 将多个文件或目录进行打包。                                   |
| -A   | 追加 tar 文件到归档文件。                                    |
| -f   | 指定包的文件名。包的扩展名是用来给管理员识别格式<br/>的，所以一定要正确指定扩展名； |
| -v   | 显示打包文件过程；                                           |

解包命令

|选项|含义|
|-|-|
|-x|对 tar 包做解打包操作。|
|-f|指定要解压的 tar 包的包名。|
|-t|只查看 tar 包中有哪些文件或目录，不对 tar 包做解打包操作。|
|-C 目录|指定解打包位置。|
|-v|显示解打包的具体过程。|

### **参考**

#### .tar文件

打包

```sh
tar -cvf <文件名>.tar <文件名>
```

解包

```sh
tar -xvf <文件名>.tar
```

#### .tar.gz文件

打包

```bash
tar -zcvf <文件名>.tar.gz <文件名>
```

解包

```bash
tar -zxvf <文件名>.tar.gz
```

#### .tar.bz2文件

打包

```bash
tar -jcvf <文件名>.tar.bz2 <文件名>
```

解包

```bash
tar -jxvf <文件名>.tar.bz2
```
